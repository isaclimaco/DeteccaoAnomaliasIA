# Projeto – Detecção de Anomalias em Folhas (IA – UnB)

Backend desenvolvido em FastAPI contendo:

- Pipeline Pix2Pix
- Pipeline ΔE2000 (CIEDE2000)
- Pipeline Grad-CAM
- Endpoints para upload de imagem e retorno de resultados

## Como rodar

Instalar dependências:

fastapi
uvicorn
torch
torchvision
pillow
numpy
colormath
opencv-python

drive pix2pix: [https://drive.google.com/drive/folders/1UrDfBv9FmNMYtZXfsyN8Och-LwynouIt?usp=drive_link
